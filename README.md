# 📊 Gestão Financeira Pessoal

Projeto desenvolvido para automatizar o tratamento de dados financeiros utilizando Google Apps Script.

## Objetivo

Transformar os registros de receitas, investimentos e despesas em uma base analítica estruturada para utilização em dashboards e análises financeiras.

## Funcionalidades

- Leitura dos dados da planilha financeira
- Consolidação de receitas, investimentos e despesas
- Padronização dos registros
- Geração automática da base analítica
- Atualização da data de processamento

## Tecnologias Utilizadas

- Google Sheets
- Google Apps Script
- Looker Studio

## Fluxo da Solução

Planilha Financeira
↓
Apps Script
↓
Base Analítica
↓
Dashboard

## Arquivo Principal

- `atualizarBaseDados.js`

## Contexto

Este script faz parte de um projeto de gestão financeira pessoal desenvolvido para automatizar a preparação dos dados e facilitar a análise das informações por meio de dashboards interativos.
